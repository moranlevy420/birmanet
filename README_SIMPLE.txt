   PENSION FUNDS EXPLORER - QUICK START
============================================

FOR WINDOWS USERS (Simple Version)
----------------------------------

FIRST TIME SETUP:
1. Double-click: INSTALL_WINDOWS.bat
2. Follow the on-screen instructions
3. If Python download page opens - install Python
   IMPORTANT: Check the box "Add Python to PATH"
4. Run INSTALL_WINDOWS.bat again after installing Python

TO RUN THE APP:
1. Double-click: run_app.bat
2. Wait for browser to open
3. Done!

TO STOP THE APP:
- Close the black command window

TO UPDATE THE APP:
- Ask the person who gave you this app for an updated folder

============================================
   TROUBLESHOOTING
============================================

Problem: "Python is not installed"
Solution: Run INSTALL_WINDOWS.bat and follow instructions

Problem: App won't start
Solution: 
1. Delete the file: pension_cache.db
2. Try running run_app.bat again

Problem: Data not loading
Solution:
1. Check your internet connection
2. Click the "Refresh Data" button in the app

Need help? Contact the person who shared this app with you.
